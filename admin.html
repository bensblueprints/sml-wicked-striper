GOOGLE CALENDAR INTEGRATION

Connect your Google Calendar so confirmed bookings automatically sync as events. When you confirm a booking, it will be added to your calendar.

Not connected

Create at Google Cloud Console → Credentials → OAuth 2.0 Client ID (Web). Add https://sml-wicked-striper.netlify.app as authorized JavaScript origin. Enable the Google Calendar API.

GOOGLE ANALYTICS

Add your Google Analytics 4 Measurement ID to track all website traffic, conversions, and user behavior in your Google Analytics dashboard.

Get your Measurement ID from Google Analytics → Admin → Data Streams → Web. It looks like G-XXXXXXXXXX. Once saved, the tracking code is automatically injected into all pages.

BOOKING EMAIL NOTIFICATIONS

Get an email every time someone submits a booking request. Notifications are sent by Netlify to the email(s) below.

This sets the Netlify form notification email. You'll receive an email with all booking details each time someone submits the form.

QUICK CALENDAR (NO SETUP NEEDED)

Don't want to set up OAuth? No problem. When you confirm a booking, an "Add to Google Calendar" button will appear. Click it to add the event directly — no API connection needed.

✓ This works by default with no setup.
